Base de Dados

Gerencie a exportação e importação dos dados do sistema.

Exportar Backup

Baixe um arquivo de backup em formato **JSON** contendo todas as suas bancas cadastradas e todo o seu histórico de apostas associado. Use este arquivo para guardar com segurança seus registros ou importá-los em outro navegador/computador.

Importar Backup

Restaurar dados de um arquivo `.json` exportado anteriormente. ⚠️ **Atenção**: Esta ação é irreversível e substituirá completamente todas as suas bancas e apostas atuais.
